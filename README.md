检举权是指公民对国家机关及其工作人员违法**失职行为**向有关国家机关予以**揭发**的权利。

# 14. 剖析辅导员的封建体制——谈如何让猪管理人

辅导员是什么？答：高校封建体制下的精神控制机器人，没有能力的跳梁小丑，从来没有出过社会的狂妄自大之徒

有能力的人不会去当高校辅导员，只有文化素质低下，品德素质差的人才会很想当辅导员,何以这么说呢？

首先呢，国内大学的体系是一级一级包围的。从学校--学院--系--专业--班级--寝室--床位号&学号，每一层都被包围在更大的一层里。小到寝室有寝室长，班级有班委和班主任，专业有辅导员，大到学院级别就有学生会检查卫生考勤和学校行政部门。这里面的大部分人行政人员都是没什么意义的人，不外乎教职工家属、大学时期就是爱搞团派的废物之流。

辅导员就在系-专业-班级这个三角地带内，管理学生的活动、学生社团、评优、思想动态、择业中国是管理下的引导，实际上根本没有必要存在。死命往上钻的，想在35岁前混个什么团委书记之流，45岁之前能当上副党委书记，最后当院党委书记或者进校级其他处级建制部门。实际上，行政级别虚高严重。他们既不搞科研，也不进行专业课的教授，只会搞点自己都不会的《就业指导》、谁上都行的《军事训练》之类。一个辅导员，自己都没去外面就业过，谈什么就业，谈什么教学生就业，完全就是外行指导内行。特别是计算机、电气、微电子学院这样的情况更严重，大部分辅导员甚至都是文科、艺术类出来的，他们怎么能教工科的就业？不如说，他们根部就不懂工科，只会掉弄一些没用的文字把戏，每天盯着领导干了什么事，赶紧写一篇报道丢到学院官网上、学院微信公众号上，就算完成基本任务了。然后，看看有什么上面传达下来的通知，建一个班委群，天天当个肉喇叭传达就行了。这么要求的话，我看让头猪来也能当辅导员。

搞出来的结果就是某些没有职位的教授不能参与治校建议，建议、决定权可能还不如刚入职的辅导员说得上话。动不动大谈理想信念，实际上只不过是想在高校混饭吃，还十分自鸣得意。这种人是如何诞生的呢？应当追溯到大学时期，他们就是这个社团，那个组织，什么学生会的狗腿子，当个狗当出了自豪。让这些人来管理，就会像”愚公移山，子子孙孙无穷“一样繁衍生息，诞生出一大批没任何对社会有意义的人。这些人整天只惟上，以讨好上级为人生最大目标，至于弄虚作假更是常规操作。更不如说，每一个辅导员都有大学时期作弊的嫌疑，都有论文抄袭的嫌疑。

从本次事件发生以来我们就能窥见一二，以情况最严重的管理、计算机、微电子学院为例，为什么他们要护着上榜的人呢？**根本原因就是因为这些学院的党委、纪委、团委里面的人本身就是一路作弊、造假上来的人。** 像类人猿生物薛原就不用多说，抄袭刘总工的论文，把里面的一张表，直接复制乱编扩充了二十多页，占了整篇硕士论文的五分之一，剩下的五分之四不是引用，就是复制粘贴。再比如说朱文拿，硕士论文完全就是抄的，查重率为100%。你说让这些人来管理学校，怎么可能查清楚有多少作弊的人呢？因为本身他们就是卑鄙、无耻的下流臭老九。

要打倒的臭老九就是这种人，正经的教职工哪个看得起他们？也就只会欺负没什么抵抗能力的大学生罢了，也就只会在出事的时候威胁学生罢了，也就只会装腔作势，活像一只只跳梁的猪。就算最后侥幸升上了学院副党委书记又怎么样？还不是一样。

当然，还有像陈曾珍这种人，没什么能力，装腔作势的能力很有，有得很嘛。动不动就是约谈举报学生，动不动威胁毕业，动不动就是来句”从来没有人像你这样诋毁“。如何评价这种人呢？应该说是党的一种悲哀，这种人就应该坚决踢出党的队伍。合工大团委怎么样？只能说是个很多自娱自乐的，自以为是的小人聚集的地方。因为高校行政结构提供的权力，就自以为是自己的话有权威，实际上让头猪上来当肉喇叭就可以达到目的。

明朝的时候先有东厂锦衣卫用以监视满朝文武，后又开设西厂多用宦官相互以制衡。如今的高校继承了东厂和西厂两个功能，何以见得呢？首先，西厂大部分的人出身是宦官，先天不足，没有后代，只能忠于朱明皇家，皇权就是他们权力的由来，所以明朝最拥护皇权的就是西厂的人。其次，东厂的主要功能就是用来进行思想监视、行为管控，督察任何有威胁皇权的行为。综上所述，高校的辅导员同时兼具了西厂的惟上权力来源结构，又负责了东厂的监视学生思想动态的功能。故而可以得出结论，辅导员的存在是一种封建制度的延续。

应该说大部分学校的学生会都没什么意义，不像北京大学、清华大学那样能诞生出锦涛总书记、春华主席这样的人，最多培养出挂职一些没啥实际意义的单位搞些无聊工作混日子的干部。**像在名单中的人，大部分就是学生会的人，或者是班委。想要一揭发就让他们受到惩罚，是一件困难的时期。** 因为，他们已经找好了保护伞，找好了庇护所，像卞子豪、刘佳欣这样的人在数学竞赛省赛作弊被举报后也只是撤销了一点学生会的职务或者班长，根本没有实际意义。依我看，所有上榜的人都要至少发一个红头文件，已经毕业的撤回毕业证，还没保研的提前取消资格。

## 15.3. 主线：冒牌神童刘端阳，AI“神棍”的传奇骗术

中国科学技术大学新创校友基金会与中国科学技术大学少年班校友会（以下分别简称“中国科大新创校友基金会”与“科大少年班校友会”，两个并列时则简称“我们”）经调查发现：莫比嗨客CEO刘端阳自称“12岁考入中国科学技术大学少年班”（“中国科学技术大学少年班”以下简称“少年班”）。经多方查实：刘端阳冒充少年班校友多年，并涉嫌假冒清华大学硕士、斯坦福大学博士（或博士生），通过莫比嗨客公司欺诈深圳、苏州、青岛、山西吕梁等多个城市与高新科技园区。

我们接校友投诉，刘端阳频繁在苏州工业园以科大少年班校友出现，疑似造假。调查结果如下，特此向社会公告。

刘端阳：“12岁考入中国科学技术大学少年班”，子虚乌有的“神童”

1) 我们查找了78级第一期少年班至今的所有学员、查询了中国科大文革之后所有入学的本科生学籍名单。未发现刘端阳。

2）我们调阅了中国科学技术大学校友（总）会公开之学籍记录，所有校友名录（不仅限于少年班学员、不仅限于本科生），未发现刘端阳。

3）我们联络中国科大档案馆、少年班学院等所有相关机构， 证实所有校友学籍未发现刘端阳。

4）关于出生年月与相关班级追访：存在“刘端阳1986年出生在宁夏固原一个农民家庭”(吕梁网）与“1984年5月出生，清华大学硕士”（“2018年度苏州工业园区金鸡湖人才表彰大会光荣册”）两种说法。因其“12岁考入中国科学技术大学少年班”，因此推测如属实，为96-99级少年班。

尽管前述所有权威部门均证实其造假，我们仍考虑刘端阳进入少年班复试、被少年班复试录取（但未报到）的极小理论可能。我们扩大延伸到咨询了95-00级少年班每个班级、多位同学；在中国科大少年班校友微信群询问情况；少年班班主任等多位知情教师询问；部分教师继续询问相近年份的科大招生办负责教师。我们在所有机构全部获得反馈，一致认为：刘端阳系假冒少年班校友。

调查核实，刘端阳曾向多位校友自称其来自中国科大98级。98级班主任与本机构负责人（碰巧1998年担任少年班复试笔试、面试监考、体检与成绩录入全程助教）均证实：无此人。

《知识分子》发现：刘端阳从未投师斯坦福大学教授Christopher Manning。他实为合肥工业大学2003级材料科学与工程金属材料二班的学生
